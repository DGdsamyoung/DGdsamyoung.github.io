过于长文本超过容器之后显示问题


word-break:normal;

在这里的话如果一些单词过长用word-break:normal;只会在单词结束后才换行，使用break-all 允许在单词内换行。


当然，你也可以使用overflow:hidden;将多余的超过容器的直接隐藏了

或者
/* 内容过长显示成省略号（内容显示为一行） */

white-space: nowrap;/设置不换行/
overflow: hidden; /设置隐藏/
text-overflow: ellipsis; /设置隐藏部分为省略号/
